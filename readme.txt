-> Code Requirements
- Opencv("pip install opencv-python")
- Tkinter("pip install Tkinter")
- PIL ("pip install Pillow")
- Pandas("pip install pandas")
-Text to speech("pip install Pyttsx3")

-> What steps you have to follow??
- Download the project folder from the drive which i have shared 
- Create a "TrainingImage" folder in a project.
- Open a "AMS_Run.py" and change the all paths with your system path
- Run "AMS_Run.py".

->Working of the project

- After run you need to give your face data to system so enter your ID and name in box than click on "Take Images" button.
- It will collect 71 images of your faces, it save a images in "TrainingImage" folder(Number of images can vary, for more accurate model trainning you can increase the number of images to be taken).
- After that we need to train the model(for train a model click on "Train Image" button.)
- The time taken to train the model depends on the number of images taken and also on the plateform on which the project is running, in my case i have done this project on system having 4GB RAM and Core i3.
- After training click on "Automatic Attendance" ,it can fill attendace by your face using our trained model (model will save in "TrainingImageLabel" )
- it will create ".csv" file of attendance according to time & subject.
- In this project Database functionality modules are coded efficiently but the no any Database has been connected due to limited time.
- "Manually Fill Attendace" Button in UI is for fill a manually attendance (without facce recognition),it's also create a ".csv" and store in a database.
